Reputifly API Engine

A headless, serverless form submission engine designed for high-performance agencies. Decouple your design from your logic.

Zero-Config Backend

Stop using PHP mailers or bloated plugins. Send a JSON payload to our endpoint, and the Reputifly Engine handles validation, spam filtering, and branded email delivery.

Quick Start

The JavaScript Protocol

Copy this snippet into your website's script area. It handles the fetch request automatically.

const handleSubmit = async (e) => {
    e.preventDefault();
    
    // 1. Define Data
    const formData = {
        _to: "client@gmail.com", // REPLACE THIS PER CLIENT
        _subject: "New Website Lead",
        name: document.getElementById('name').value,
        email: document.getElementById('email').value,
        message: document.getElementById('message').value
    };

    // 2. Send to Engine
    const response = await fetch('https://reputifly-5bcmmm5hbq-as.a.run.app', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(formData)
    });

    if (response.ok) {
        alert("Sent successfully!");
        e.target.reset();
    }
};

Endpoint & Schema

POST https://reputifly-5bcmmm5hbq-as.a.run.app
Field Type Required Description
_to String YES Target email address (The Client).
name String YES Lead's full name.
email String YES Lead's email address (Reply-To).
message String YES The inquiry body content.
_avatar URL NO Optional URL to user profile picture.
