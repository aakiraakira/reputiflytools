CMS User Guide

Welcome to the Sunny Real Estate management system. Use this guide to update your inventory and market insights.

1. How to Add a Property

Follow these steps to add a new listing to your main inventory.

1
Create New Entry

Go to your WordPress Dashboard sidebar and click Properties > Add New.

2
Basic Information

Title: Enter the name of the building or unit (e.g., "Suntec City Tower 2").

Featured Image: (Right Sidebar) Upload the main cover image here. This is the first image users see.

3
Property Data (ACF Fields)

Scroll down to the "Property Details" section and fill in the following:

Field Name Input Type Description
Price * Number The raw number (e.g. 5000000). Do not include "$" or commas.
Size (Sqft) * Number Total area in square feet (e.g. 2500).
Listing Type * Select Choose Buy or Rent. This controls which filter it appears under.
Category * Select Must match system IDs: office, warehouse, retail, shophouse, business_park.
Region Select central, east, west, etc.
Address Text Short address displayed on card (e.g. "123 Robinson Road").
Status Select new (Default), promotion (Cyan Tag), leased (Dark Tag), sold.
Gallery Gallery Upload additional images here. These appear in the Fullscreen Modal.

2. How to Add a Blog Post

Use this for Market Updates, News, or Insights. The system will automatically apply the "Full Page Takeover" design.

1
Create Article

Go to Dashboard > Posts > Add New.

2
Set the "Hero" Image

Important:

You MUST set a Featured Image in the right sidebar. This image becomes the fullscreen background header for the article.

3
Write Content

Use the standard WordPress editor to write your text.

  • The Title becomes the large headline.
  • The Date is added automatically.
  • A "Chat on WhatsApp" button is added automatically at the bottom.
