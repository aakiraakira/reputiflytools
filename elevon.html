How to Update the Website

1

Generate the File

Fill out the form in the "Create Job" tab and click Download. This will save a small file to your computer (e.g., marketing-manager.json).

2

Go to the Data Folder

Login to GitHub and navigate to: src > data > jobs.

3

Upload the File

Click the Add file button (top right) and select Upload files.

Drag and drop the file you just downloaded into the box.

4

Save Changes

Scroll down to the green button labeled Commit changes and click it.

To Delete a Job: Simply go to the same folder, click on the specific job file, click the trash icon, and "Commit changes".
