How to Upload to Website

1

Generate the File

Fill out the form in the "Create Job" tab and click Download. This will save a small file to your computer (e.g., 2025-marketing-manager.json).

2

Go to the Job Database

Click the button below to open the specific folder on GitHub where jobs are stored.

Open GitHub Jobs Folder →
3

Upload the File

On the GitHub page:

  1. Click the Add file button (top right).
  2. Select Upload files.
  3. Drag and drop the file you just downloaded into the box.
4

Publish Changes

Scroll down to the green button labeled Commit changes and click it. The website will update automatically in 1-2 minutes.

To Delete a Job: Click the link in Step 2, click on the specific job file you want to remove, click the Trash Icon (üóëÔ∏è), and "Commit changes".
